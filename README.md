### Up & Running

Install dependencies:
```sh
$ yarn
```

Start application:
```sh
$ yarn start
```

### Tests

Run the below command to run integration/unit tests:
```sh
$ yarn test
```

Run the below command to run E2E tests:
```sh
$ yarn cy:run
```

__ps: _don't forget to start the app in order to run E2E tests!___
